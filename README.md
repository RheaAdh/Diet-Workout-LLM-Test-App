# Diet and workout plan using Hugging Face

## User Input and Preferences
- Age
- Gender
- Height
- How many days do you workout
- Veg/Non-Veg
- Dietry restrictions

## Features
- BMI (P1)
- Diet plan (P1)
- Workout plan (P1)

## Tech
- LLM Model : Hugging Face
- Frontend : React.js
- Database: Mysql
- Backend : FastAPI

## Run Server
- uvicorn main:app --reload
